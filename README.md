Login Page UI Demo

Files created:
- index.html — main UI (open in browser)
- styles.css — styles for the page
- script.js — small UI interactions (modal and form stubs)

How to try:
1. Open a terminal in `c:/Users/wes/OneDrive/Desktop/App Prodjects/counting`.
2. Run `npm install`.
3. Run `npm start`.
4. Open `http://localhost:3000` in your browser.

Notes:
- Login is saved to the browser so you do not need to sign in again on the same device.
- Each account stores saved games in `accounts.json`, including current games and past rounds.
- Add players one at a time, then start the game.
- The scoreboard arranges players left-to-right, shows score history under each player, and totals at the bottom.
- This is a demo app; game and account data are saved locally in the project folder.
